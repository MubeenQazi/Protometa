- 🌍 SEO-Friendly
- 📱 100% Responsive Design
- ✉️ Supports Contact Form
- 📊 Google Analytics support
- 💬 Disqus Comment Support
- 🖼️ Custom Logo Support
- 👥 Multi-Author

- 🎨 Includes All SCSS Files
